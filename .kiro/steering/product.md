# Product Overview

子供向けのルールに縛られない自由なリバーシ（オセロ）ゲーム。
リアルタイム対戦機能を備え、ルームIDを共有して友達と遊べる。

## Core Capabilities

- **2フェーズターン制**: 配置フェーズ（通常リバーシルール）→ 裏返しフェーズ（相手の石を自由に裏返し、回数無制限）
- **自動パス**: 相手に合法手がない場合は自動的にパス
- **リアルタイム対戦**: WebSocketによる盤面の即時同期（計画）
- **ルームベースのマッチング**: ルームID共有方式（ユーザー登録不要）
- **直感的なUI**: 子供が迷わず操作できるシンプルなインターフェース

## Target Use Cases

- 友達や家族とオンラインでオセロを遊ぶ
- 通常ルールに縛られず自由に石を操作して遊ぶ（イカサマOK）
- 子供がルールを覚える前の練習として使用

## Value Proposition

**「アナーキーモード」**という独自コンセプト:
- 配置は通常リバーシルール（挟める場所に石を置く）
- 配置後の裏返しフェーズで相手の石を自由に何枚でも裏返し可能
- ターン制で交互にプレイ、手番終了ボタンで交代
- 勝敗よりも「遊ぶ楽しさ」を優先した設計

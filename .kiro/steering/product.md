# Product Overview

子供向けのルールに縛られない自由なリバーシ（オセロ）ゲーム。
リアルタイム対戦機能を備え、ルームIDを共有して友達と遊べる。

## Core Capabilities

- **自由な石操作**: クリックで石を自由に配置・反転（黒→白→空のサイクル）
- **リアルタイム対戦**: WebSocketによる盤面の即時同期
- **ルームベースのマッチング**: ルームID共有方式（ユーザー登録不要）
- **直感的なUI**: 子供が迷わず操作できるシンプルなインターフェース

## Target Use Cases

- 友達や家族とオンラインでオセロを遊ぶ
- 通常ルールに縛られず自由に石を操作して遊ぶ（イカサマOK）
- 子供がルールを覚える前の練習として使用

## Value Proposition

**「アナーキーモード」**という独自コンセプト:
- 通常のオセロと違い、合法手の判定を行わない
- どのマスでも自由にクリックして石を操作可能
- ターン制限なし、誰でもいつでも操作可能
- 勝敗よりも「遊ぶ楽しさ」を優先した設計

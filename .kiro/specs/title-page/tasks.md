# Implementation Plan

## Task 1: Home ページをタイトルページに改修

- [x] 1. Home ページをタイトルページに改修
- [x] 1.1 ルーム作成・参加UIを削除し、シンプルなタイトルページへ変更する
  - ルーム作成ボタン、ルーム参加フォーム、`roomId` の状態管理、`handleCreateRoom`・`handleJoinRoom` ハンドラーを削除する
  - ゲームタイトル「アナーキーオセロ」をヘッダーエリアに `h1` で表示する
  - タイトル下にキャッチコピー（例: 「自由に石を操作できるリバーシゲーム」）を表示する
  - アナーキーモードの説明文（配置フェーズ・裏返しフェーズの概要）をタイトルエリア内に表示する
  - タイトルテキストをモバイル・デスクトップ両対応でレスポンシブに大きく表示する
  - _Requirements: 1.1, 1.2, 1.3, 1.4, 2.3_

- [x] 1.2 「ゲームを始める」ボタンを実装する
  - 画面中央に「ゲームを始める」ボタンを1つだけ目立つ形で表示する
  - ボタンクリックで `/room/local` へ `navigate` で遷移する
  - キーボード操作（Enter/Space キー）でもゲームを開始できるようにする
  - ボタンに `role="button"`, `tabIndex={0}`, `aria-label` を設定する
  - ボタンのタッチターゲットを最低44×44pxとする
  - フォーカス時に視覚的なフォーカスインジケーター（`focus:outline-2 focus:outline-yellow-400`）を表示する
  - _Requirements: 2.1, 2.2, 2.4, 3.1, 3.2, 3.3_

- [x] 1.3 ビジュアルデザインを整える
  - グラデーション背景（`from-blue-50 to-indigo-100` 相当）を維持・適用する
  - ダークモード対応のカラーパレットを適用する
  - ページ全体をビューポート中央に縦横センタリングして表示する
  - フッターエリアに補足テキスト（「子供向け自由度の高いリバーシゲーム」等）を表示する
  - _Requirements: 4.1, 4.2, 4.3, 4.4_

## Task 2: テスト

- [x] 2. テスト
- [x] 2.1 表示要素のレンダリングテストを作成する
  - タイトル「アナーキーオセロ」が `h1` で表示されることを検証する
  - キャッチコピーが表示されることを検証する
  - アナーキーモード説明文が表示されることを検証する
  - フッターテキストが表示されることを検証する
  - ルーム作成・参加関連のUI要素が表示されないことを検証する
  - _Requirements: 1.1, 1.2, 1.3, 2.3, 4.4_

- [x] 2.2 ゲーム開始ナビゲーションとアクセシビリティのテストを作成する
  - 「ゲームを始める」ボタンクリックで `navigate('/room/local')` が呼ばれることを検証する
  - Enter キー押下でゲームが開始されることを検証する
  - Space キー押下でゲームが開始されることを検証する
  - ボタンに `role="button"` が設定されていることを検証する
  - ボタンに `tabIndex={0}` が設定されていることを検証する
  - ボタンに `aria-label` が設定されていることを検証する
  - _Requirements: 2.1, 2.4, 3.1, 3.2, 3.3_
